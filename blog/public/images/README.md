# Cover images for blog posts

Add **1200×630 px** images here (JPG or PNG, under ~300 KB).

Reference in post frontmatter:

```yaml
coverImage: "/blog/images/your-post-cover.jpg"
coverImageAlt: "Describe the image for accessibility and SEO"
coverCaption: "Optional caption under the hero image"
```

Files in `blog/public/` are served at `/blog/...` on huntmode.ca.
